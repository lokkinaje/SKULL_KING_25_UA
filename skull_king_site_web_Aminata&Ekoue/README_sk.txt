
Projet de Concrétisation – Skull King
======================================

Réalisé par : Aminata Tako BA et Ekoue Kliko  – L1 Informatique

Description rapide :
--------------------
Ce site présente notre projet de programmation du jeu Skull King en Python.
Il contient :
- Une page d’accueil
- Une page pour présenter notre binôme
- Une page expliquant les règles du jeu
- Une page pour présenter notre code (structure et explications)

Structure des fichiers :
------------------------
index_sk.html       → page d’accueil
binome_sk.html         → présentation de l’équipe
regles_sk.html         → règles du jeu
code_sk.html           → explication du code

Conseils :
----------
- Ouvrir d’abord le fichier : index_sk.html
- Naviguer ensuite avec le menu (ou directement dans les pages)
- Si certains styles ne s’affichent pas : vérifier que les fichiers CSS sont bien dans le dossier
- Site testé sur Google Chrome et Firefox

Merci pour la lecture !
